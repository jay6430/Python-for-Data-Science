Seeing data patterns using scatterplots

Scatterplots show clusters of data rather than trends (as with line graphs) or discrete values (as with bar charts). The purpose of a scatterplot is to help you see data patterns.

polyfit examples

polyfit takes the input of 2 arrays or shape (x[i],y[i]) and result into vector of coefficients p that minimises the squared error. see error formula on official site

Vectors are just arrays with a single column.

import numpy as np
import pandas as pd

import datetime as dt

print(df)
